## 📦 Features

- ⚡ Real-time collaborative editing  
- 🧠 Conflict-free sync using Yjs  
- 🔄 Live updates with Socket.IO  
- 💻 Monaco Editor (VS Code-like experience)  
- 🌐 Full-stack (React + Express + Node.js)
